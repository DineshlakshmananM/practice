C Data Types

What are Data Types?

Data types specify the type and size of data that a variable can hold. C provides several built-in data types that determine how much memory is allocated and what operations can be performed on the data.

Basic Data Types

1. int: Integer values (whole numbers)
2. float: Floating-point numbers with single precision
3. double: Floating-point numbers with double precision
4. char: Single character or ASCII value
5. void: Represents absence of type

Integer Data Type

int age = 25;              // Standard integer (2-4 bytes)
short count = 100;         // Smaller range (2 bytes)
long population = 1000000; // Larger range (4-8 bytes)
unsigned int score = 500;  // Non-negative values only

printf("%d\n", age);        // Print integer

Floating-Point Data Types

float price = 19.99;       // Single precision (4 bytes)
double pi = 3.14159265;    // Double precision (8 bytes)

printf("%f\n", price);      // Print float with 6 decimals
printf("%.2f\n", price);    // Print float with 2 decimals

Character Data Type

char grade = 'A';          // Single character
char newline = '\n';       // Escape sequence
int ascii_value = 65;      // 'A' has ASCII value 65

printf("%c\n", grade);      // Print character
printf("%d\n", (int)'A');   // Print ASCII value

Size of Data Types

printf("int: %lu bytes\n", sizeof(int));
printf("float: %lu bytes\n", sizeof(float));
printf("double: %lu bytes\n", sizeof(double));
printf("char: %lu bytes\n", sizeof(char));

Type Modifiers

short: Reduces the size of integer
long: Increases the size of integer
signed: Can hold positive and negative values (default)
unsigned: Can hold only non-negative values

Variable Declaration and Assignment

int x, y, z;               // Multiple declarations
int a = 10, b = 20;        // Declaration with initialization
float salary = 5000.50;
char initial = 'J';

// Rules for variable names:
// - Must start with letter or underscore
// - Can contain letters, digits, underscores
// - Case-sensitive (age != Age)
