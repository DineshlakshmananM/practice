C Control Statements

What are Control Statements?

Control statements are used to control the flow of program execution. They determine which instructions to execute and how many times to execute them based on certain conditions.

if Statement

Executes a block of code if a condition is true:

int age = 18;

if (age >= 18) {
    printf("You are an adult\n");
}

if-else Statement

Executes one block if condition is true, another if false:

int age = 15;

if (age >= 18) {
    printf("You are an adult\n");
} else {
    printf("You are a minor\n");
}

if-else if-else Statement

int score = 75;

if (score >= 90) {
    printf("Grade: A\n");
} else if (score >= 80) {
    printf("Grade: B\n");
} else if (score >= 70) {
    printf("Grade: C\n");
} else {
    printf("Grade: F\n");
}

switch Statement

Used when you have multiple alternatives:

int choice = 2;

switch(choice) {
    case 1:
        printf("You chose option 1\n");
        break;
    case 2:
        printf("You chose option 2\n");
        break;
    default:
        printf("Invalid choice\n");
}

while Loop

Repeats as long as condition is true:

int i = 0;

while (i < 5) {
    printf("%d\n", i);
    i++;
}

do-while Loop

Executes at least once, then repeats if condition is true:

int i = 0;

do {
    printf("%d\n", i);
    i++;
} while (i < 5);

for Loop

Repeats a fixed number of times:

// for (init; condition; update)
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}

break and continue

break: Exits the loop immediately
continue: Skips the rest of current iteration
for (int i = 0; i < 10; i++) {
    if (i == 3) continue;      // Skip 3
    if (i == 7) break;          // Exit loop at 7
    printf("%d\n", i);
}
