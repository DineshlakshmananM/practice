C Functions

What is a Function?

A function is a reusable block of code that performs a specific task. Functions help organize code, make it more readable, and allow code reuse. Every function has a name, parameters, and a return type.

Function Basics

Return Type: Data type of value returned by function
Function Name: Unique identifier for the function
Parameters: Input values passed to function
Function Body: Code that executes when function is called
Return Statement: Value sent back to caller

Function Declaration (Prototype)

Tells compiler the function exists before it is defined:

// Prototype
int add(int a, int b);

// Later in code
int result = add(5, 3);

Function Definition

Actual implementation of the function:

int add(int a, int b) {
    int sum = a + b;
    return sum;
}

Function with No Return Value

Use return type void when function doesn't return anything:

void greet(char name[]) {
    printf("Hello, %s!\n", name);
    // No return statement needed
}

// Calling the function
greet("John");

Function with No Parameters

int getCurrentYear() {
    return 2024;
}

int year = getCurrentYear();

Complete Function Example

// Function that calculates factorial
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

// Calling the function
int result = factorial(5);      // Result: 120

Function Parameters

Pass by Value: Copy of variable is passed, original not changed
Pass by Reference: Address of variable is passed using pointers, can modify original

Scope of Variables

  • Local Variables: Declared inside function, only accessible there
  • Global Variables: Declared outside functions, accessible everywhere
  • Static Variables: Retain value between function calls

Recursive Functions

A function that calls itself:

// Calculate sum of numbers 1 to n
int sumNumbers(int n) {
    if (n == 0) {
        return 0;           // Base case
    }
    return n + sumNumbers(n - 1);   // Recursive call
}
