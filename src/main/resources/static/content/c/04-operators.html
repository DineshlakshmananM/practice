C Operators

Introduction to Operators

Operators are symbols that perform operations on variables and values. C supports various types of operators for performing arithmetic, logical, relational, and bitwise operations.

Arithmetic Operators

+ Addition
- Subtraction
* Multiplication
/ Division (integer division if both are integers)
% Modulus (remainder after division)

Arithmetic Operations Example

int a = 20, b = 10;

printf("%d\n", a + b);      // Output: 30
printf("%d\n", a - b);      // Output: 10
printf("%d\n", a * b);      // Output: 200
printf("%d\n", a / b);      // Output: 2
printf("%d\n", a % b);      // Output: 0

Assignment Operators

int x = 10;
x += 5;                     // x = x + 5;  (x is now 15)
x -= 3;                     // x = x - 3;  (x is now 12)
x *= 2;                     // x = x * 2;  (x is now 24)
x /= 4;                     // x = x / 4;  (x is now 6)

Relational Operators

== Equal to (returns 1 if true, 0 if false)
!= Not equal to
> Greater than
< Less than
>= Greater than or equal to
<= Less than or equal to

Logical Operators

int a = 1, b = 0;

// && (AND) - Both conditions must be true
if (a && b) { printf("Both true\n"); }  // Not executed

// || (OR) - At least one condition must be true
if (a || b) { printf("At least one true\n"); }  // Executed

// ! (NOT) - Reverses the result
if (!b) { printf("b is false\n"); }     // Executed

Increment and Decrement

int count = 5;

count++;                    // Post-increment (count becomes 6)
++count;                    // Pre-increment (count becomes 7)
count--;                    // Post-decrement (count becomes 6)
--count;                    // Pre-decrement (count becomes 5)

Operator Precedence

Operators are evaluated in this order (highest to lowest):
1. () [] (Parentheses, Brackets)
2. ++ -- + - ! (Increment, Decrement, Unary)
3. * / % (Multiplication, Division, Modulus)
4. + - (Addition, Subtraction)
5. < <= > >= (Relational)
6. == != (Equality)
7. && (Logical AND)
8. || (Logical OR)
9. = += -= etc. (Assignment)
