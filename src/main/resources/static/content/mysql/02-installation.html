MySQL Installation and Setup

System Requirements

Before Installing MySQL, You Need:
• Operating System: Windows, macOS, or Linux
• RAM: Minimum 512 MB (1 GB recommended)
• Disk Space: 100 MB minimum for installation
• Administrator/Root Access: To install system-wide

Download MySQL

Visit the official MySQL website at https://www.mysql.com/downloads/ and select the MySQL Community Server (free version).

Installation on Windows

1. Download the MySQL installer (.msi file)
2. Run the installer and choose setup type:
   - Developer Default (full installation)
   - Server only
   - MySQL Router only
3. Configure MySQL Server (port 3306, UTF8)
4. Configure MySQL as a Windows service
5. Set root password during configuration
6. Finish installation

7. Verify installation:
   Open Command Prompt and type:
   mysql --version
   mysql -u root -p (enter root password)

Installation on macOS

Using Homebrew (recommended):
1. Install Homebrew if not already installed
2. Run: brew install mysql
3. Run: brew services start mysql
4. Secure installation: mysql_secure_installation
5. Set root password and remove anonymous users

Verify installation:
mysql --version
mysql -u root

Installation on Linux (Ubuntu/Debian)

1. Update package list:
   sudo apt update

2. Install MySQL Server:
   sudo apt install mysql-server

3. Run the security script:
   sudo mysql_secure_installation

4. Set root password and security options

5. Start MySQL service:
   sudo systemctl start mysql

6. Verify installation:
   mysql --version
   sudo mysql -u root -p

Verify MySQL Installation

Check MySQL version:
mysql --version

Connect to MySQL server:
mysql -u root -p

Inside MySQL shell:
SHOW VARIABLES LIKE 'version';
EXIT;

MySQL Workbench (GUI Tool)

MySQL Workbench is a visual tool for managing MySQL databases. Download from https://www.mysql.com/products/workbench/

  • Visual database design
  • SQL script editor
  • Database administration
  • Data migration tools
