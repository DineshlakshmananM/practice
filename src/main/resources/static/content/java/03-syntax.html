Java Syntax Basics

Java Syntax Overview

Java syntax refers to the set of rules that define the structure and writing style of Java programs. Understanding proper syntax is crucial for writing valid Java code.

Basic Program Structure

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Key Java Keywords

public: Accessible from anywhere
class: Defines a class
static: Can be called without creating an instance
void: Method returns nothing
String[]: Array of strings for command-line arguments
System.out.println(): Prints to console

Variables and Data Types

int age = 25;              // Integer
double salary = 50000.50;  // Decimal number
boolean isActive = true;   // Boolean value
char grade = 'A';          // Single character
String name = "John";      // String (sequence of characters)

Naming Conventions

  • Class names: Start with uppercase (HelloWorld, Person)
  • Variable names: Start with lowercase, use camelCase (firstName, studentAge)
  • Constants: All uppercase with underscores (MAX_USERS, PI)
  • Methods: Verb names in camelCase (calculateTotal, printMessage)

Declaring and Initializing Variables

// Declaration
int x;

// Initialization
x = 10;

// Declaration with initialization
int y = 20;

// Multiple variables
int a = 5, b = 10, c = 15;

Type Casting

Converting one data type to another:

// Implicit casting (automatic, no data loss)
int intValue = 42;
double doubleValue = intValue;

// Explicit casting (manual, possible data loss)
double decimal = 9.99;
int integer = (int) decimal;  // Result: 9

Output to Console

System.out.println("Hello");     // Prints with newline
System.out.print("Hello");       // Prints without newline
System.out.printf("%d %.2f\n", 42, 3.14);  // Formatted output

Java Operators

Arithmetic: + - * / %
Assignment: = += -= *= /=
Comparison: == != < > <= >=
Logical: && || !
