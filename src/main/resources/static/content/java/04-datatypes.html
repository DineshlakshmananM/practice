Java Data Types

What are Data Types?

Data types in Java specify the size and type of values that a variable can hold. Java supports two categories of data types: primitive types and reference types.

Primitive Data Types

byte: 8-bit integer (-128 to 127)
short: 16-bit integer (-32,768 to 32,767)
int: 32-bit integer (default for whole numbers)
long: 64-bit integer
float: 32-bit single precision decimal
double: 64-bit double precision decimal (default for decimals)
char: 16-bit Unicode character
boolean: true or false

Integer Data Types

byte smallNumber = 100;
short mediumNumber = 5000;
int standardInt = 100000;
long largeNumber = 10000000000L;  // L suffix for long

Floating-Point Data Types

float price = 19.99f;           // f suffix for float
double salary = 50000.75;       // double is default
double pi = 3.14159265358979;

Character and Boolean

char letter = 'A';
char digit = '5';
char unicode = '\u0041';  // Unicode character

boolean isJavaFun = true;
boolean isProgrammer = false;

Reference Data Types (Non-Primitive)

  • String: Sequence of characters
  • Array: Collection of elements of same type
  • Class: User-defined reference type
  • Interface: Contract that classes can implement

String Data Type

String name = "John Doe";
String message = "Hello, Java!";
String empty = "";

// String methods
int length = name.length();           // Get length
String upper = name.toUpperCase();    // Convert to uppercase
String lower = name.toLowerCase();    // Convert to lowercase
char firstChar = name.charAt(0);      // Get character at index

Arrays

// One-dimensional array
int[] numbers = {10, 20, 30, 40, 50};
String[] names = new String[3];

// Accessing array elements
int first = numbers[0];      // 10
names[0] = "Alice";

// Array length
int size = numbers.length;   // 5

Type Conversion

// Automatic conversion (widening)
int intValue = 10;
long longValue = intValue;

// Manual conversion (narrowing)
double doubleValue = 9.99;
int intFromDouble = (int) doubleValue;  // Result: 9

// String to number
int num = Integer.parseInt("42");
double decimal = Double.parseDouble("3.14");

Variable Declaration

int age = 25;
double height = 5.9;
String country = "USA";
boolean isStudentActive = true;

// Multiple variables of same type
int x = 5, y = 10, z = 15;
