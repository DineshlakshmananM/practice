Java Introduction

What is Java?

Java is a general-purpose, object-oriented programming language designed to be portable and platform-independent. It follows the philosophy: "Write Once, Run Anywhere" (WORA).

History of Java

Created by: Sun Microsystems (1995)
Creator: James Gosling
Current Owner: Oracle Corporation (since 2010)
Latest LTS Version: Java 21 (released September 2023)

Key Features of Java

  • Simple: Similar syntax to C/C++ but simplified
  • Object-Oriented: Everything is an object
  • Platform-Independent: Bytecode runs on any JVM
  • Secure: Built-in security features
  • Robust: Strong memory management and exception handling
  • Multithreaded: Built-in support for concurrent programming

Java Platform Architecture

Java Virtual Machine (JVM): Executes Java bytecode
Java Runtime Environment (JRE): JVM + libraries for running Java programs
Java Development Kit (JDK): JRE + tools for developing Java programs

Your First Java Program
