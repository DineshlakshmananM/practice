Java Installation and Setup

System Requirements

  • Operating System: Windows, macOS, or Linux
  • RAM: Minimum 2GB (4GB or more recommended)
  • Disk Space: 100MB for JDK, more for IDE
  • Internet: Required for downloading JDK and dependencies

What to Install

JDK (Java Development Kit): Contains JRE, compiler (javac), and development tools
IDE (Optional): IntelliJ IDEA, Eclipse, or NetBeans for easier development
Text Editor (Optional): VS Code, Sublime Text as lightweight alternative

Download Java JDK

Visit: https://www.oracle.com/java/technologies/downloads/

  • Download latest LTS version (Java 21 LTS recommended)
  • Select installer for your operating system
  • Accept license terms and download

Installation on Windows

  1. Run the downloaded .exe file
  2. Click "Next" to proceed through installation wizard
  3. Choose installation directory (default is fine)
  4. Complete installation
  5. Open Command Prompt and verify: java -version

Installation on macOS

  1. Run the downloaded .dmg file
  2. Double-click installer package
  3. Follow installation prompts
  4. Open Terminal and verify: java -version

Installation on Linux

# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-21-jdk

# Fedora/CentOS
sudo dnf install java-21-openjdk-devel

# Verify installation
java -version

Verify Installation

Check if installation was successful:

java -version          # Shows Java version
javac -version         # Shows compiler version
echo $JAVA_HOME         # Shows Java installation path

Install an IDE (Recommended)

IntelliJ IDEA Community: Professional IDE, free community edition
Eclipse: Open-source, lightweight, widely used
VS Code: Lightweight, requires extensions
NetBeans: User-friendly for beginners

Set Up First Project

  1. Create a folder for your project
  2. Create a file named HelloWorld.java
  3. Write your first program
  4. Compile: javac HelloWorld.java
  5. Run: java HelloWorld
✅ Next Step: Once installed, write and run your first Java program to confirm everything works!
