Python Setup

Installing Python

Windows

  1. Visit python.org/downloads
  2. Download the latest Python installer (.exe)
  3. Run the installer
  4. Important: Check "Add Python to PATH"
  5. Click "Install Now"

macOS

Option 1 - Homebrew:
brew install python
Option 2 - Official Installer:
Download from python.org/downloads

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install python3 python3-pip

Verifying Installation

python --version
python3 --version

Choosing an IDE

VS Code (Recommended for Beginners)

  1. Install Visual Studio Code
  2. Install Python extension
  3. Install Pylance extension for better IntelliSense

Other Options

  • PyCharm: Powerful IDE specifically for Python
  • Jupyter Notebook: Great for data science
  • Thonny: Beginner-friendly IDE
