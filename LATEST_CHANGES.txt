- Added authentication check to `learning.html`
- Users without login session are redirected to login page
- Shows alert: "Please login first to access learning content"

**Code Added:**
```javascript
window.addEventListener('DOMContentLoaded', function() {
    const user = localStorage.getItem('user');
    if (!user) {
        alert('Please login first to access learning content');
        location.href = 'login.html';
        return;
    }
});
```
